COMMON ISSUES & QUICK FIXES
───────────────────────────────────────────────────────────────────────────────

Issue: "Python is not recognized"
Fix:   Install Python from https://www.python.org/
       ✓ CHECK "Add Python to PATH" option
       ✓ RESTART computer after install
       ✓ Then try again: python --version

Issue: "ModuleNotFoundError: No module named 'androguard'"
Fix:   Run: python -m pip install androguard
       Or let the script install it (it will if missing)

Issue: Script runs but output folder is empty
Fix:   1. Run: python verify_setup.py (check for errors)
       2. Verify source has XML files
       3. Check you have write permissions
       4. Try again

Issue: "permission denied" or "access denied"
Fix:   Run Command Prompt as Administrator
       Right-click cmd.exe → Run as administrator

Issue: Script seems to freeze or be very slow
Fix:   Normal for 1000+ files - can take 5+ minutes
       Check CPU in Task Manager to verify it's working
       Only cancel if no CPU activity for 10+ minutes

TECHNOLOGY EXPLAINED
───────────────────────────────────────────────────────────────────────────────

What's happening under the hood:

Binary XML (In APK):
  • Compressed Android proprietary format
  • Uses string pools and type references
  • Optimized for runtime performance
  • Size: ~1.2 KB per file (example)
  • Can't be read directly

Decoder Process (androguard library):
  • Reads binary XML header
  • Extracts string pool
  • Parses namespace information
  • Processes element definitions
  • Resolves attribute values
  • Reconstructs standard XML

Text XML (After decoding):
  • Standard XML format
  • All values resolved and visible
  • Ready for analysis/editing
  • Size: ~3.8 KB per file (example, more verbose)
  • Human-readable
